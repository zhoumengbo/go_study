# go_study

从零开始学习go语言，因为go本身是内存安全的，适用于分布式系统的学习。并且Docker、k8s这些工具都是go开发的。

1. 语法简洁；Go语言简单易学，学习曲线平缓。
2. 代码风格统一。
3. 执行性能好。
4. 开发效率高；Go语言内置了垃圾回收机制，使其拥有像Python、PHP等解释性语言的一样的开发效率。
5. 支持并发。有封装好的RPC包和goroutine的支持。
